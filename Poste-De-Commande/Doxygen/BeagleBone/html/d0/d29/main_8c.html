Detailed Description

This file's purpose is to handle a terminal communication between the CommandCenter.exe program and a beagleBoneBlue. it parses what is received in the terminal and outputs CAN once a proper data tram is received.

Data trams are as followed:
0xFE/ADR1/ADR2/ADR3/'M'/Mode/C1/command1/C2/command2/'S'/State
Each time a data tram is sent, [-SYNC-] is shown on the terminal.
