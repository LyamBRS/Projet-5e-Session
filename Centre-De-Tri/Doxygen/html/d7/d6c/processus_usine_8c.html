◆ delaiModeDeTest()

bool delaiModeDeTest ( bool  reset)

Cette fonction, appelée de façon récurrente, compte jusqu'à VALEUR_DELAI_MODE_DE_TEST, puis retourne un 1 cela permet de faire un délai de 2 sec pour plusieurs des étais du mode de tests sans copier coller un gros bout de code à chaque fois.

Paramètres
resetest mis a DELAITEST_RESET pour mettre le délai à zéro ou est mis à DELAITEST_UTILISE pour uliliser le délai.
Renvoie
true si délai écoulé
false si délai en cours
