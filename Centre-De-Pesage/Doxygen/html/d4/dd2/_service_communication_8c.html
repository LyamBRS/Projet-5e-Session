Takes CircularCommandsBuffer and parses it into the CAN transmission buffer. It checks for the first AVAILABLE slots in the latter and puts the first element IN_QUEUE at that index. ◆ CircularCommandsBuffer

unsigned char CircularCommandsBuffer[24]

This buffer is used to store which commands or values were QUEUE by he program, to send them x by x amount each time the CAN slot is equal to your module's appropriated slot.
Do not use this buffer outside of this file. The program handles it by itself.
You need 24 bytes of memory to store this buffer.

If no value is stored in at an index, it will be 0xFF.
Otherwise, the value of the stCommands is stored in it.
