Description détaillée

Processus qui gere le véhicule du projet de 5eme session de TSO. Le xprocessusVehicule.h gere les requetes de transport de rondelles envoyé par les autres modules du projet. Il utilise le ServiceCommunication.h ainsi que sa structure ModuleData pour savoir quand conduire les rondelles.
