Le processus Conduite est une phase du service base de temps qui est appeler à chaques périodes de la base de temps. Cela permet de démarrer une requête de conduite depuis un autre processus quand il est nécessaire. Le processus conduite a pour but de suivre la ligne noir installé au sol pour le projet de 5e session de TSO.
